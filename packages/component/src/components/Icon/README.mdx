# 图标 Icon

组件库内部的图标组件，用于展示组件库内必要的图标。其他图标可以使用 [unocss icon](https://unocss.dev/presets/icons) 和 [mdi](https://pictogrammers.com/library/mdi/)

</code>

<code src="./__demo__/fontSize.tsx" title='icon尺寸'  >

</code>

<code src="./__demo__/spin.tsx" title='旋转' propsController={{spin:true}}>

</code>
